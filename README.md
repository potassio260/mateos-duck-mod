# Minecraft Forge 1.20.1 Duck Mod

As the duck lover I am, Minecraft's chickens never really did the job. Neither did many of the other mods out there, which served as a huge inspiration for this project.

This mod completely overhauls your poultry experience. It handles all kinds of duck mechanics, including spawning, animations, duck variants, and genders with specific skins. It features secret Easter eggs, new items and blocks, and up to 13 different custom AI goals (5 of which are custom-tailored specifically for these ducks).

## Features

* **Immersive Ducks:** Fully animated ducks with custom skins, distinct genders, and personalized gender-based breeding.
* **Custom Behaviors:** 13 unique AI goals, custom attacks, and a specialized method to tame them as pets.
* **Natural Spawning:** Ducks spawn naturally in Plains, River, and Forest biomes, featuring a unique, turtle-like natural reproduction mechanic with a twist.
* **New Items & Blocks:** Introduces custom spawn eggs, layable eggs, customizable duck nests, and the ability to cook duck meat.

## Tech Stack

* **Language:** Java
* **Framework:** Minecraft Forge 1.20.1
* **Modeling & Animation:** Blockbench *(Standard for custom entities)*

## Installation & Setup

*(Work in Progress)*

*Note: Forge mods for Minecraft Java Edition are distributed as `.jar` files, not `.apk` files (which are for Bedrock/Android).*

Once fully released, players can install the mod using these standard steps:
1. Download and install [Minecraft Forge for 1.20.1](https://files.minecraftforge.net/net/minecraftforge/forge/index_1.20.1.html).
2. Download this mod's `.jar` release file.
3. Place the downloaded `.jar` file into the `%appdata%\.minecraft\mods` folder.
4. Launch Minecraft using the Forge 1.20.1 profile.
